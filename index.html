Commit a change

git commit -m "Commit message"
  • If you don't supply -m it will start $EDITOR and will finish the commit when you quit
  • -v Will let you see the diff of the changed about to be committed

Creating a branch

git branch branch-name
  • 'git branch -a' to see all branches
  • 'git branch -d|-D branch-name' to delete the branch
  • 'git branch -m|-M branch-name' to rename the branch
[any material that should appear in print but not on the slide]

Merging a branch

git merge branch-name
  • --squash If you want all the commits from the branch to be merged in one commit

Making a tag

git tag tag-name
  • -l see all tags
  • -d delete a tag. Do not do this if other people have pull the tag
