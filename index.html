Creating A Branch

git branch branch-name
  • 'git branch -a' to see all branches
  • 'git branch -d|-D branch-name' to delete the branch
  • 'git branch -m|-M branch-name' to rename the branch

Understanding rev-parse / Tree-ish

HEAD~n | n commits behind HEAD
{yesterday}, {1 month 2 weeks 3 days 1 hour 1 second ago} or {1979-02-26 18:30:00} | Time based
branch@{n} | n commits behind branch
  • http://www.kernel.org/pub/software/scm/git/docs/git-rev-parse.html
